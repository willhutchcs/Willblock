
# Willblock

Do you LOVE basketball but HATE advertisements? Do you watch NBA streams ethically from paid subscription services? Well, well, well, my friend, Willblock is the program for you.

Willblock is a real-time ad-blocker that utilizes AI trained on various games and advertisements from NBA broadcasts. All the training data was collected from official reuploads on YouTube so you can go to sleep at night knowing your ad-block is 100% morally grounded! 



## How Does It Work?? Is It Magic? Are You A Magic Man With Special powers???

It might *seem* like it's witchcraft, but I promise it's nothing of the sort! The python script simply takes a screenshot of your desktop every few seconds and if it detects an ad, it'll display an overlay with music! You can use your own overlay/music, but I've included mine for FREE! How neat is that?


## Installation

To run on your own machine you need the following packages:
```bash
  pip install pywin32 tkinter torch torchvision keyboard threading python-vlc pyopengl pygetwindow pycaw comtypes ctypes PIL
```

This program is designed to capture Firefox. If you want to use another broswer/application you can manually change it in the script. I plan on creating an easy way to manage that later.
*If you're using a web browser, you have to disable Hardware Acceleration for the program to work.*
